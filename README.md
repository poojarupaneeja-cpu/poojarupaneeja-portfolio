# Pooja Rupaneeja - Interactive Portfolio

A modern, interactive portfolio website showcasing content writing, brand strategy, and creative work.

## 🚀 Features

- **Smooth Animations**: Scroll-triggered animations and micro-interactions using Framer Motion
- **Responsive Design**: Perfect on desktop, tablet, and mobile devices
- **Filterable Projects**: Filter portfolio work by category (Copywriting, Social Media, SEO, etc.)
- **Interactive Navigation**: Sticky nav with active section highlighting
- **Contact Form**: Direct messaging and social links
- **Performance Optimized**: Fast loading with Next.js
- **Modern Design**: Black & beige theme with premium aesthetic

## 🛠️ Tech Stack

- **Next.js 14** - React framework for production
- **React 18** - UI library
- **Framer Motion** - Animation library
- **Tailwind CSS** - Utility-first CSS framework
- **TypeScript** - Type safety
- **Lucide React** - Icon library

## 📦 Installation

1. Clone the repository
```bash
git clone https://github.com/poojarupaneeja-cpu/poojarupaneeja-portfolio.git
cd poojarupaneeja-portfolio
```

2. Install dependencies
```bash
npm install
```

3. Run the development server
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## 🎨 Customization

### Update Content
- Edit component files in `/components` folder
- Update project data in `components/Projects.tsx`
- Modify contact information in `components/Contact.tsx`

### Customize Colors
- Edit color variables in `tailwind.config.ts`
- Current theme: Black (#000000) & Beige (#F5F1E8)

### Add Portfolio Links
- Replace Google Drive links in `components/Projects.tsx` with your actual portfolio pieces

## 📝 Sections

1. **Hero** - Eye-catching introduction
2. **About** - Bio and expertise
3. **Projects** - Filterable portfolio showcase
4. **Contact** - Contact form and social links
5. **Footer** - Quick navigation and information

## 🚀 Deployment

Deploy easily to Vercel:

```bash
npm install -g vercel
vercel
```

Or connect your GitHub repo to Vercel for automatic deployments.
